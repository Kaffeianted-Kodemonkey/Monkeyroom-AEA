A11y Evaluation

All evaluations are conducted using our A11y Evaluation Audit tool (AEA)
to recorded test results generated the Audit Report.

Understanding it's unrealistic to conduct a manual test on every page of a large site. We suggest reviewing these key pages to start:

  • Site templates
  • Interactive forms
  • Dynamic content pages
  • Pages with dialog modals and alert notifications
  • Key entry and exit pages (including account login and recovery pages)
  • Help and assistance pages

Then submit both the json and html files to AEA-Complete.

Evaluation Test

  1. Click Start Evaluation button below
  2. Select Choose File and load a JSON file:
    • First time use the SuccessCriteriaTT4.json
    • Returning use previously created JSON report file.
  3. Select Load File button.
  4. Open Product and Testing Information to complete each form and the required fields.
  5. Add/Edit Test Results.
  6. Select Save button or the keyboard shortcut (Alt+S).
Start Evaluation

Audit Report

  1. Select Audit Report button below.
  2. Select Choose File and load JSON file with test results.
  3. Select Open.
  4. Select Load Results button.
  5. View the Test Results.
  6. Select the Save as HTML button at the bottom or use Alt+S to generate a printer-friendly version of the report and save JSON file for future use.
