To ensure smooth progress and efficient collaboration all interaction will be done via email or LinkedIn Messenger.

For the evluation testing of all sites, we use the following assistive technologies and tools:

  • ANDI a free open-source bookmarklet designed to help test web content for accessibility.
  • Color Contrast Analyzer (CCA) an open-source tool displaying the contrast ratio for two selected colors. CCA is available at the following links:
    • CCA 2.5 for Windows
    • CCA 2.4 for macOS
  • WAVE Web accessibility browser plug-in
  • Screen reader: NVDA, Jaws
  • Mobile Devises: iSO, Andriod
  • Acrobat: CommonLook PDF plug-in for evaluating PDF's
